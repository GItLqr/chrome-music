## 介绍
音乐电台chrome扩展，包含豆瓣电台等14个电台。

该插件提供了目前最流行的音乐电台播放，包括：百度听、豆瓣电台、酷狗电台、虾米电台、酷我电台、乐酷电台、多米音乐、音悦电台、亦歌电台、人人电台、百度mp3音乐盒、搜狗音乐盒、Google音乐盒。


## 安装

使用chrome浏览器进入 http://goo.gl/L18RM 页面，点击安装即可。
